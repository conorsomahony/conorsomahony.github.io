The Mobile and Device Development module involved building Mobile web applications using third party client side JavaScript frameworks and libraries such as React.js, jQuery and jQuery Mobile.

Project: React.js Web Application
The Brief

"Create a React.js application that provides To-Do List style functionality. Your app should include a form to add To-Do items, a list of items, dynamically populated by user data and the ability for the user to remove items. Your app should save its own state to localStorage. To-Do List items should persist if the app is restarted."
